# Path tracing project
C++ implementation made for College, improved for fun.

Features:

- Importance sampling
- Post-processing (denoising, bloom)
- Multicore execution
- Acceleration structures
- Signed distance fields rendering
- Subsurface Scattering

## Code ##
I became contributor of [Yocto/GL](https://github.com/xelatihy/yocto-gl/tree/development), moved the project there.
